## What is OMFK?

You're typing, deep in thought... then you look up:

```
Ghbdtn? rfr ltkf&   →   Привет, как дела?
ру|дщ цщкдв          →   hello world
```

Wrong keyboard layout. Again.

**OMFK fixes this automatically, as you type.**

## Features

- **EN / RU / HE** keyboard layouts
- **On-device ML** — your typing never leaves your Mac (CoreML)
- **Auto-correction** — fixes as you type
- **Manual override** — Option key to cycle alternatives

## Requirements

- macOS 14.0+
- Apple Silicon or Intel

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/chernistry/omfk-releases/releases)
2. Open the DMG and drag OMFK to Applications
3. Launch OMFK — grant Accessibility permission when prompted
4. Done! OMFK runs in your menu bar
